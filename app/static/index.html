Stake Integration
✓ Token saved — valid ~30 days

Used only to sync your Stake portfolio — never shared.

▸How to get your session token
Stake uses a request header called Stake-Session-Token — find it in the Network tab (not Cookies). You only need to do this once; the token is valid for ~30 days.
Chrome / Edge (desktop)
  1. Log in at app.hellostake.com
  2. Press F12 → go to the Network tab
  3. In the filter box type api.stake to narrow results
  4. Refresh the page — you'll see API requests appear
  5. Click any request (e.g. equities/positions)
  6. In the right panel click Request Headers
  7. Find Stake-Session-Token — copy the full value
  8. Paste it into the field above and save
Safari (Mac)
  1. Enable DevTools: Safari → Settings → Advanced → tick "Show Develop menu"
  2. Log in at app.hellostake.com
  3. Develop → Show Web Inspector → Network tab
  4. Filter by api.stake, refresh, click any request → Headers → find Stake-Session-Token
The token expires after ~30 days. Re-paste a fresh one when sync stops working.
